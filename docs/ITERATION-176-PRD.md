# Iteration 176 - 产品需求文档 (PRD)

## 产品分析

### 现有功能完成度评估
| 模块 | 完成度 | 用户价值 |
|------|--------|----------|
| 像素办公室场景 | 95% | 核心体验 |
| 多Agent状态监控 | 90% | 高 |
| Cron任务面板 | 85% | 高 |
| 心情系统 | 80% | 中 |
| 番茄钟 | 85% | 中 |
| 成就系统 | 75% | 中 |
| 资产管理 | 90% | 中 |
| 昨日日记 | 70% | 中 |

### 用户痛点分析
1. **缺乏数据洞察** - 有状态监控但无历史趋势，用户无法了解工作模式
2. **通知能力弱** - 仅番茄钟有浏览器通知，Cron失败/Agent异常无主动提醒
3. **日记体验单一** - 仅显示昨日日记，无法浏览历史，无法搜索
4. **缺乏环境氛围** - 像素办公室是静态的，缺少时间感和环境变化
5. **交互反馈不足** - 状态切换无音效，操作缺乏即时感

## 本次迭代需求 (P0-P2)

### P0: 工作数据统计面板
**目标**: 让用户直观了解Agent的工作规律和效率趋势

**功能点**:
1. **今日工作时间线** - 横向时间轴显示今日各状态的时间分布（色块条）
2. **本周工作统计** - 7天柱状图，显示每天各状态占比
3. **Agent活跃度排行** - 按Token消耗/会话数排序的Top Agent列表
4. **Cron健康度指标** - 成功率百分比 + 最近7天趋势迷你图

**后端需求**:
- `GET /stats/today-timeline` - 返回今日状态变更时间线
- `GET /stats/weekly` - 返回7天统计数据
- 从state.json历史和openclaw sessions数据聚合

**前端需求**:
- 新增 `stats-panel.js` 模块
- 纯CSS实现柱状图和时间线（不引入图表库）
- 面板放置在Cron面板下方

---

### P0: 日记浏览器增强
**目标**: 让日记从"只看昨天"升级为可浏览的工作日志

**功能点**:
1. **日期翻页** - 左右箭头切换日期，显示对应日期的日记
2. **日记列表** - 显示最近7天有日记的日期，点击切换
3. **日记内容增强** - 支持markdown基础渲染（加粗、列表、代码块）

**后端需求**:
- `GET /memo/list` - 返回所有可用日记日期列表
- `GET /memo/:date` - 返回指定日期的日记内容
- 复用现有 `MEMORY_DIR` 和 `extract_memo_from_file`

**前端需求**:
- 改造现有memo-card区域，添加翻页控件
- 简易markdown渲染（正则替换，不引入库）

---

### P1: 实时通知中心
**目标**: 关键事件不遗漏，Cron失败、Agent上线等主动推送

**功能点**:
1. **通知铃铛图标** - 右上角显示未读数量角标
2. **通知列表面板** - 点击展开最近20条通知
3. **通知类型**:
   - Cron任务失败（红色警告）
   - Agent状态变更（上线/离线）
   - 番茄钟完成
   - 成就解锁
4. **浏览器推送** - 关键通知触发 Notification API

**实现方式**:
- 前端本地通知队列（不需要后端存储）
- 从现有轮询数据中检测变化并生成通知
- 新增 `notifications.js` 模块

---

### P1: 办公室氛围系统
**目标**: 让办公室有"活的"感觉，增强沉浸感

**功能点**:
1. **日夜循环** - 根据本地时间自动切换办公室色调
   - 6:00-8:00 晨光（暖黄滤镜）
   - 8:00-17:00 白天（正常）
   - 17:00-19:00 黄昏（橙色滤镜）
   - 19:00-6:00 夜晚（深蓝滤镜 + 窗户灯光效果）
2. **状态粒子效果** - 不同状态显示不同粒子
   - writing: 飘落的纸片
   - error: 红色警告闪烁
   - syncing: 数据流光线
   - researching: 放大镜光圈
3. **咖啡机动画** - idle状态时咖啡机冒蒸汽

**实现方式**:
- Phaser tint/filter 实现日夜滤镜
- Phaser particles 实现粒子效果
- 新增场景后处理层，不修改现有sprite逻辑

---

### P2: 快捷键系统
**目标**: 键盘快速操作，提升效率用户体验

**快捷键映射**:
| 按键 | 功能 |
|------|------|
| 1-6 | 切换对应状态 |
| P | 番茄钟 开始/暂停 |
| M | 切换心情选择器 |
| N | 打开/关闭通知面板 |
| D | 打开/关闭装修抽屉 |
| ? | 显示快捷键帮助 |
